Emirates Stadium Arsenal Football Club 60 260 22 juillet 2006

L'Emirates Stadium, référencé sous le nom d'Arsenal Stadium pour les compétitions de l'UEFA, est un stade de football situé dans le nord de Londres, à Highbury dans le borough londonien d'Islington. Arsenal Football Club, club de Premier League, y évolue depuis juillet 2006.
